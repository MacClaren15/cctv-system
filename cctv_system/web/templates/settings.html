📹 Camera Settings

Frame Rate (FPS) 30

Adjust the number of frames per second. Higher values = better quality but more CPU usage. Lower values = smoother performance.

Resolution

Higher resolution improves quality but requires more processing power.

Brightness 50

Adjust video brightness. 0 = very dark, 100 = very bright.

Contrast 50

Adjust contrast between light and dark areas. 50 = normal, higher = more contrast.

Saturation 50

Adjust color saturation. 0 = grayscale, 50 = normal, 100 = very vibrant.

⚡ Motion Detection Settings

Motion Detection

Enable or disable motion detection system.

Detection Algorithm

MOG2: Best overall, detects moving objects. Frame Diff: Fastest, detects any change. Optical Flow: Most accurate motion direction.

Sensitivity 50

Lower = More sensitive (catches small movements), Higher = Less sensitive (prevents false detections). Recommended: 30-60.

Motion Threshold 500

Minimum number of pixels that must change to trigger motion detection.

Motion Alert Enabled

Enable alerts when motion is detected.

🎥 Recording Settings

Auto Recording

Automatically record video when motion is detected.

Recording Duration (seconds) 30

How long to record each motion detection event.

Keep Recordings (days) 7

Automatically delete recordings older than this many days.

Recording Storage Path (USB Drive Primary)

Path to store recordings (USB drive recommended). Examples: /media/usb (Linux), /mnt/usb (Linux), C:\USB (Windows), D:\ (Windows). Leave empty to use default (./data/recordings).

Fallback Storage

If USB drive is unavailable, automatically save to default directory (./data/recordings).

Sound Notifications

Email Notifications

Email Address

Alert Cooldown (seconds) 30

Minimum time between alerts to prevent spam. (30 seconds prevents 2+ alerts per minute)
